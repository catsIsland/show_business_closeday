# README

### カレンダーで定休日を表示できる機能になります。

#### 手順
- アカウント登録 (現在は非表示)
- 発行されたタグをご利用したいページに設定
- 管理画面で定休日またはそれ以外の日程を選択
- カレンダーを表示したい要素のid値を設定
- 背景色、文字色を設定(defaultあり)
- 公開

### 現在利用中のページ
うえすとこーすと：https://westcoast-jp.com/
